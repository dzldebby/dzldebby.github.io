BUILDING A VOLUNTEER-STUDENT MATCHING PLATFORM

    Full Stack Developer

PROJECT OVERVIEW

Developed for MAD WISH initiative by It's Raining Raincoats, this platform streamlines the matching of migrant workers in Singapore with volunteer English-language mentors. The application automates the previously manual matching process, making it more efficient and scalable while ensuring optimal student-teacher pairings.

PROJECT WALKTHROUGH

KEY FEATURES

  • Secure authentication system with role-based access control
  • Intuitive interface for student and teacher profile management
  • Automated matching system with status tracking
  • Comprehensive screening process for volunteers
  • Native language compatibility tracking

TECHNICAL ARCHITECTURE

FrONTend (Vue.js)

  • Vuex for centralized state management
  • Vue Router for seamless navigation
  • Buefy UI framework for consistent design
  • Modular architecture for scalability

Backend (NOde.js)

  • RESTful API with Express.js
  • PostgreSQL for reliable data storage
  • JWT-based authentication
  • Environment-based configuration

DEVELOPMENT PROCESS

As part of TechLadies Bootcamp #6, our team followed an agile development methodology. We worked closely with stakeholders from MAD WISH to understand their requirements and iteratively developed features. The project emphasized both technical excellence and social impact, aligning with TechLadies' mission of empowering women in technology while serving the community.

IMPACT

  • Streamlined volunteer-student matching process
  • Reduced administrative overhead for coordinators
  • Improved tracking and management of teaching relationships
  • Enhanced volunteer screening and verification process

LEARNING OUTCOMES

This project provided valuable experience in:

  • Building full-stack applications with modern frameworks
  • Implementing secure authentication systems
  • Working in an agile development environment
  • Collaborating with stakeholders to deliver social impact
